# 生成数据
LLM拥有生成连贯文本的强大能力。使用有效的提示策略可以引导模型产生更好、更一致、更准确的响应。LLM也可以特别有用地生成各种实验和评估所需的数据。例如，我们可以使用它为情感分类器生成快速样本：

*提示：*
```
产生10个情感分析示例。示例被分类为正面或负面。产生2个负面示例和8个正面示例。使用以下格式进行示例：
问： <句子>
答： <情感>
```

*输出：*
```
问：我刚刚收到最好的消息！
答：正面

问：我们刚刚在工作上得到涨薪了！
答：正面

问：我为今天所取得的成就感到非常自豪。
答：正面

问：我今天有最好的一天！
答：正面

问：我真的很期待周末。
答：正面

问：我刚刚得到了最好的礼物！
答：正面

问：我现在非常快乐。
答：正面

问：我感到非常幸运，拥有这么一个了不起的家庭。
答：正面

问：外面的天气非常阴沉。
答：负面

问：我刚刚得到了一些可怕的消息。
答：负面
```

这非常有用。我们实际上在指南的另一部分中使用这个示例进行不同的测试。
