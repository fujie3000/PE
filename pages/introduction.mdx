# 介绍

提示工程是一门相对较新的学科，旨在开发和优化提示，以有效地使用语言模型（LM）来进行各种应用和研究。提示工程技能有助于更好地了解大型语言模型（LLMs）的能力和限制。研究人员使用提示工程来提高LLMs在各种常见和复杂任务上的能力，如问题回答和算术推理。开发人员使用提示工程设计健壮而有效的提示技术，与LLMs和其他工具接口。

本指南涵盖了提示的基础知识，以提供如何使用提示与大型语言模型（LLMs）交互和指导的大致想法。

除非另有说明，否则所有示例都使用`text-davinci-003`进行测试（使用OpenAI的播放器）。 它使用默认配置，即`temperature = 0.7`和`top-p = 1`。
