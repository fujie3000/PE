# 提示工程指南

提示工程是一个相对较新的学科，旨在开发和优化提示以有效地使用语言模型（LM）进行各种应用和研究主题。提示工程技能有助于更好地理解大型语言模型（LLM）的能力和局限性。

研究人员使用提示工程来提高LLM在各种常见和复杂任务上的能力，例如问题回答和算术推理。开发人员使用提示工程来设计鲁棒且有效的提示技术，以与LLM和其他工具交互。

提示工程不仅仅是设计和开发提示。它涵盖了与LLM交互和开发所需的各种技能和技术。这是一项重要的技能，用于接口，构建和了解LLM的能力。您可以使用提示工程来提高LLM的安全性，并构建新的能力，例如使用领域知识和外部工具增强LLM。

受到对LLM开发的高度兴趣的推动，我们创建了这个新的提示工程指南，其中包含所有最新的论文，学习指南，模型，讲座，参考资料，新的LLM能力和与提示工程相关的工具。
