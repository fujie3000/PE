[Yao et al., 2022](https://arxiv.org/abs/2210.03629) 引入一种框架，LLMs 以交织的方式生成推理轨迹和任务特定的操作。生成推理轨迹使模型能够诱导、跟踪和更新行动计划，甚至能处理异常情况。操作步骤允许与外部来源（如知识库或环境）进行接口交互和信息收集。

ReAct 框架可以让 LLMs 与外部工具交互，以检索导致更可靠和真实响应的其他信息。
