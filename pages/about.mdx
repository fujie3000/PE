# 关于

Prompt Engineering Guide 是 [DAIR.AI](https://github.com/dair-ai) 的项目，旨在教育研究人员和从业者有关提示工程的知识。

DAIR.AI 旨在民主化人工智能研究、教育和技术。我们的使命是让下一代人工智能创新者和创造者能够拥有机会。

我们欢迎社区的贡献。请查看编辑按钮。

许可证信息在[这里](https://github.com/dair-ai/Prompt-Engineering-Guide/blob/main/LICENSE.md)。

我们从许多开放的资源中汲取了灵感，例如 [OpenAI CookBook](https://github.com/openai/openai-cookbook)、[Pretrain, Prompt, Predict](http://pretrain.nlpedia.ai/)、[Learn Prompting](https://learnprompting.org/) 等。
